#**Список проектов:**
1. ApiGateway (ещё не реализован обмен писем между авторизованными пользователями)
2. MessageLibrary (содержит DTO сообщений, который передаётся между сервисами)
3. MessageService (сервис для обмена сообщениями между авторизованными пользователями)
4. UserService (сервис пользователей. Если пользователи будут пользоваться напрямую 
   этим сервисом, а не через ApiGateway, то они смогут отправлять друг другу сообщения 
   только будучи авторизованными)
5. UserServiceTests (содержит в себе unit тесты сервисов)
